How Insertion Sort Works

        

          Insertion Sort treats the left side of the array as sorted. It picks one element from the unsorted side and inserts it into the correct position inside the sorted side. The most important part is the while loop, because that is where elements are shifted to make space for the key.
        

        

          Simple thought: Pick one element, move left, shift bigger elements right, and finally place the picked element at the empty position.
        

      


      

        
Complexity

        

          
Best CaseO(n)

          
Average CaseO(n²)

          
Worst CaseO(n²)

        

        

          Best case happens when the array is already sorted. Worst case happens when the array is reverse sorted, because every element has to shift many positions.
