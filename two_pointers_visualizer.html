Two Pointers Visualizer

Learn the classic Two Sum in a sorted array pattern using two pointers. One pointer starts from the left, the other from the right. Based on the current sum, we move exactly one pointer inward. This is one of the cleanest two-pointer problems for beginners.

Two pointers idea here:
If sum is too small, move left pointer right.
If sum is too large, move right pointer left.
Because the array is sorted, each move is meaningful.
